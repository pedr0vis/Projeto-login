# PROJETO LOGIN

## Descrição

<h4> A partir das aulas do professor <a href="https://github.com/leonardossrocha">Leonardo Rocha</a>, nossa turma do 3° bimestre está introduzindo o que aprendemos para criar uma página de login dinâmica com FAQ, além de aprendermos a documentar um README no github.</h4>

## Funcionalidades do projeto
- `Funcionalidade 1` `Login`: Para realizar o acesso é necessário utilizar um e-mail de sua preferência e a senha
- `Funcionalidade 2` `FAQ`: O FAQ tira dúvidas frequentes de usuários que tem dificuldade para realizar o login

<h4> Em breve será adicionado mais funcionalidades como maneiras de acesso rápido (conta google, outlook, etc), termos de serviço, etc </h4>
